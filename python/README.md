# pyinfMOSTRO

## Instalación

### Requerimientos
* Python>=3.7.0
* pip>=19.3.1

### Dependencias
`pip install -r requirments.txt`

## Compilación

`python -m compileall -b .`

## Uso
Ejecutar `main.pyc` y seleccionar un archivo de configuración `*.yml` válido de acuerdo a la documentación.
